# 0.1.7 — 2026-05-24

## Changed

- **`openlibrary-author` / `openlibrary-work` resource handlers** — removed catch-all `try/catch` wrappers that converted timeout, network, and upstream errors into a generic `NotFound`. Errors now propagate with their correct classification.
- **`getEditionByIdentifier`** — author name lookups parallelized with `Promise.all` instead of sequential `for...of` await.
- **`extractDescription`** — removed unnecessary `as` casts; TypeScript narrows correctly after the `'value' in raw` guard.
- **`getAuthor` remote_ids spread** — simplified conditional spread from `{ ...( x ? { k: v } : {}) }` to `...( x != null && { k: v } )` pattern.
- **`invalid_identifier` error code** — corrected from `InvalidParams` to `ValidationError`; ISBN/OLID format checks are domain validation, not malformed RPC input.
- **Skills** — `git-wrapup` v1.0 added; `release-and-publish` v2.5 → v2.6.
- **`CLAUDE.md`** — added "Close the loop on issues" rule; framework version reference updated to `^0.9.9`.
- **`@cyanheads/mcp-ts-core`** ^0.9.7 → ^0.9.9 (fuzz pre-parse fix, `fast-check` moved to optional peer dep, `git-wrapup` skill shipped, `ctx.fail` / notify methods documented).
