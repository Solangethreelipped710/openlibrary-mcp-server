# openlibrary-mcp-server

Book metadata, search, and reading availability via the Open Library (Internet Archive) API.

## Data source

- **Open Library API** — book metadata, search, author info, editions, covers
- **Auth**: None required
- **Rate limits**: Generous, public API

## Why it earns its keep

Different niche from the academic paper servers (arXiv/PubMed/Crossref cover research papers — this covers books). 20M+ editions of books. Useful for librarians, educators, readers, researchers, and any agent that needs to look up a book.

## Target users

- Educators building reading lists
- Researchers looking up book references
- Librarians and catalogers
- Readers searching for books by topic/author
- Agents needing ISBN/OCLC/LCCN resolution

## Scope

- Read-only
- Full-text book search
- Book/edition lookup by ISBN, OCLC, LCCN, Open Library ID
- Author search and metadata
- Subject/topic browsing
- Cover image URLs
- Reading availability (borrow/read online status)
- Work → edition relationships
